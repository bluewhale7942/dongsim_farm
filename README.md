# dongsim_farm
동심팜 환경제어기 개발 관련 공유 파일입니다.

본 프로젝트는 ESP32-S3의 WebServer 기능 활용 가능성을 검토하기 위한 실험용 예제입니다.
실제 현장에 적용할 경우에는 반드시 충분한 안전 검증 및 보호 설계를 선행하시기 바랍니다.

온·습도 센서 값은 현재 랜덤 데이터로 구성되어 있습니다.
→ 실제 센서 연동을 위해서는 별도의 코드 구현이 필요합니다.

열림/닫힘 모터 출력 제어 시, 반드시 인터락(상호 배타 제어) 로직을 적용해야 합니다.
→ 동시 동작 방지를 위한 필수 안전 조건입니다.

각 제어 버튼에 대한 자동 운전 로직은 포함되어 있지 않습니다.
→ 현장 조건에 맞는 제어 알고리즘을 추가로 설계 및 적용해야 합니다.
